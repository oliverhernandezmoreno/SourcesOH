1. First, if you dont have the Storybook-addon-material-ui, then install this addon:
```
npm i storybook-addon-material-ui --save-dev
```

2. Start Storybook with:
```
npm run storybook
```

https://storybook.js.org/docs/guides/quick-start-guide/
